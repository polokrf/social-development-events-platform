
# Project Title

 #title:  Social Development Events Platform,

 # Description 
             "This is a project that showcases various social events."
## Features


1. Users can easily register or log in to the website.
2. You can join various social development organizations if you wish.
3. You can showcase your upcoming events here.
4. You can participate in different events.
5. All your data or information will be securely saved, and you can update or delete it anytime.
6. Your personal or confidential information will remain private and visible only to you.


## Tech Stack

**Client:** React,  TailwindCSS

**Server:** firebase,nodejs ,express,mongodb
